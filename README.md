## Descripción del Proyecto
Global Seguros es una plataforma web diseñada para la comercialización de seguros educativos, dirigida principalmente a padres de familia y tutores que buscan asegurar y planificar el futuro académico de sus hijos. La página ofrece una experiencia de navegación fluida donde los usuarios pueden explorar la marca, conocer detalladamente las coberturas y acceder a servicios personalizados.

Técnicamente, el proyecto se lleva a cabo mediante una interfaz accesible desarrollada con HTML5 y CSS3. La experiencia de usuario se gestiona con un carrusel interactivo de planes en la página principal y resolviendo el reto de la aseguradora mediante un simulador lógico. Este componente procesa datos y manipula el DOM en tiempo real, traduciendo variables complejas en proyecciones financieras inmediatas que generan confianza y transparencia.

## Requisitos mínimos
El proyecto está construido con tecnologías web nativas, por lo que no requiere de entornos de ejecución como Node.js ni instalaciones complejas. Para que la página funcione correctamente solo se necesita:
* Un navegador web moderno (Google Chrome, Mozilla Firefox, Safari, Microsoft Edge, etc.).
* Conexión a internet activa (indispensable para la correcta carga de las imágenes alojadas en https://sirv.com/).

## Recursos y Herramientas
* **Lenguajes:** HTML5, CSS3 y JavaScript.
* **Control de Versiones:** Git y GitHub.
* **Gestión de Recursos Multimedia:** Las imágenes oficiales suministradas por la empresa se sirven y gestionan a través de **Sirv** para optimizar la carga del sitio.
